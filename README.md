# Computational Supplement

This directory contains the computational files for the manuscript

> Ramified \(p,q\)-smooth Pell equations and 2-class groups of
> \(\mathbb{Q}(\sqrt{pq})\) for \(p \equiv q \equiv 5 \pmod 8\).

## Files

- `code/verify_theorem_magma.m`: Magma script used to verify the stated
  class-group, unit-norm, quartic-symbol, and ramified-prime principality
  criteria over the bounded range described in the manuscript.
- `outputs/magma_verification_output.txt`: recorded output from the Magma run.

## Repository

The public GitHub copy of these files is available at
<https://github.com/chatchawanpan-dev/biquadratic-pell-computations>.

## Computational Environment

The recorded run used Magma V2.29-6 with Magma's default class-group proof
settings and no conditional proof flag set in the script.

## Scope

The computations are verification and illustration only. The proofs in the
manuscript do not depend on the computational output.
